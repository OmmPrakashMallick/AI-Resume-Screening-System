# AI-Resume-Screening-System
AI-based system that analyzes resumes and compares them with job descriptions using NLP and cosine similarity to calculate candidate match scores.
## Technologies Used
- Python
- PyPDF2
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## Project Workflow
1. Upload Resume (PDF)
2. Extract text from resume using PyPDF2
3. Load job description
4. Convert text into TF-IDF vectors
5. Calculate similarity using cosine similarity
6. Generate resume match score
